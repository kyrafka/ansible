# 🔧 Scripts de NAT64

Scripts para configurar y diagnosticar NAT64/DNS64.

- **install-nat64-tayga.sh**: Instala Tayga NAT64
- **install-squid-proxy.sh**: Instala Squid Proxy
- **install-jool-nat64.sh**: Instala Jool NAT64
- **fix-nat64-routes.sh**: Corrige rutas NAT64
- **configure-nat64-dns64.sh**: Configura NAT64 y DNS64
