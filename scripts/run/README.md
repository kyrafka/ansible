# 🚀 Scripts de Ejecución Rápida

Scripts para ejecutar playbooks específicos.

- **run-network.sh**: Ejecuta playbook de red
- **run-dhcp.sh**: Ejecuta playbook de DHCP
- **run-dns.sh**: Ejecuta playbook de DNS
- **run-firewall.sh**: Ejecuta playbook de firewall
- **run-storage.sh**: Ejecuta playbook de almacenamiento
- **run-common.sh**: Ejecuta playbook común
- **run-role.sh**: Ejecuta un rol específico
- **run.sh**: Script de ejecución general
