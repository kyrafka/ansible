#  Scripts de Diagnóstico

Scripts para verificar el estado del sistema.

- **check-nat64-status.sh**: Verifica estado completo de NAT64
- **test-govc-connection.sh**: Prueba conexión con ESXi
- **test-network-connectivity.sh**: Prueba conectividad de red
- **test-ssh-ubpc.sh**: Prueba conexión SSH al servidor
