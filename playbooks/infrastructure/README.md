# 📁 Playbooks de Infraestructura

Playbooks para configurar el servidor base.

## Playbooks disponibles:

- **setup-complete-infrastructure.yml**: Configura TODO el servidor (red, DHCP, DNS, firewall, NAT64)
- **playbook-network.yml**: Solo configuración de red
- **playbook-dhcp.yml**: Solo DHCP IPv6
- **playbook-dns.yml**: Solo DNS + DNS64
- **playbook-firewall.yml**: Solo firewall
- **playbook-storage.yml**: Solo almacenamiento NFS
